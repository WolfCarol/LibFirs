firs.h是这个项目的最初版本，目前已经不再使用，取而代之的是fir.h，这是极致精简后的库，仅有15条函数，提供了windows平台开发游戏所需的最基础的功能————创建窗口、获取键盘鼠标输入、绑定OpenGL渲染、提交缓冲区播放音频。Fir_CSBind.cs是C#的绑定版本，需要提前编译fir.h为动态链接库。

=====以下为旧版本简介=====

FirsLib是一个极简的2D游戏开发库，包装了一些win32api，功能有限，只完成了一些不得不和系统打交道的活，编译成动态链接库只有30kb不到，我用来给c#做p/invoke绑定的，因为我实在不愿意继续用c/cpp开发游戏。我不确定是否有bug，只简单测了一些，我会在后续实际项目开发中使用这个库，并且修复漏洞和更新一些必要的新功能。

这个库的主要功能有以下几个方面：

1创建和管理窗口（win32api）

2获取键盘/鼠标/手柄输入（xinput）

3用opengl绘制图像

4用xaudio2播放音频

如果有人想要使用或者从这个库中学习，我非常欢迎，我声明了MIT License和Public Domain两种许可方式，我无心处理许可授权和法律纠纷。

我并不熟悉linux也没用过macos，因此我无力支持除了windows以外的平台。

这个库不会比现有的成熟的游戏/多媒体开发库更好，但是我偏爱它的小巧。

我以我的挚友的名字为这个库命名，在此我要向她表示感谢。
